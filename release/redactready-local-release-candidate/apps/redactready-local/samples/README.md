# RedactReady Synthetic Fixtures

These files contain fake identifiers for detector and UI testing. They are not real personal records.

- `sample-sensitive.txt`: mixed personal, case, financial, and secret-like values.
- `sample-claims.csv`: spreadsheet-style rows for CSV replacement testing.

The browser demo also exposes public one-click samples from `public/samples/`:

- `redactready-synthetic-contact.txt`
- `redactready-synthetic-invoice.csv`
- `redactready-synthetic-case-notes.txt`

All sample values are synthetic and exist only to exercise local review, detection, export, and reporting behavior.
