# FocusForge – Prompt Templates

This file contains example system and user prompts for each FocusForge mode.

## Planner Mode
- System prompt: define role, tone, and constraints for breaking large goals into tasks.
- User examples: how a user might describe their week or project.

## Summarizer Mode
- System prompt: emphasize brevity, clarity, and surfacing next steps.

## Accountability Mode
- System prompt: supportive, non-judgmental tone, focused on gentle nudges and reflection.
