# Astra — Case Study

## Summary

Astra is a local AI chat workspace that demonstrates AI application interface design, model configuration flow, Markdown rendering, transcript export, and practical UX around API-key configuration.

## Portfolio Value

Astra supports the portfolio by showing practical AI-app implementation rather than abstract AI discussion.

## Key Capabilities

- Chat workspace
- Express streaming API direction
- Gemini configuration
- Markdown rendering
- Transcript export
- Settings panel
- Clear configuration status

## Future Improvements

- Add provider abstraction.
- Add saved conversations.
- Add retrieval over uploaded notes.
- Add stronger error handling and tests.
