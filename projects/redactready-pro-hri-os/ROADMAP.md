# Roadmap

## Near Term

- Robust PDF text extraction with graceful browser compatibility checks.
- OCR for screenshots and scanned forms.
- Stronger address, name, and ID detection with locale-aware confidence.
- Browser print layout refinements for polished PDF export.
- More sample packets and portfolio screenshots.

## Mid Term

- Browser-based QR/barcode detection.
- Signature and face-region detection for image workflows.
- Encrypted opt-in local storage with clear delete/reset controls.
- Advanced evidence graph with timelines, claims, and source confidence.
- Optional AI summarization through a provider abstraction that is disabled by default.

## Long Term

- Collaboration and review workflows.
- Browser extension for privacy leak auditing before upload/share.
- Desktop app packaging.
- Template-specific packet preparation for common administrative workflows.
- Audit logs for redaction decisions.
