# Project Index

This index records public names, source authority, implementation status, demos, and the best reason to review each project. It avoids treating every project as equally mature or applying one universal ranking across employer, technical, research, and buyer audiences.

## Status and Authority Legend

- **Portfolio workspace** — runnable source is present under `apps/` in this repository.
- **Separate authoritative repository** — current product development and release evidence are maintained in another repository.
- **Local-only** — runnable source is present, but no current public production demo is claimed.
- **Documentation-first** — case study or specification exists without a maintained `apps/` workspace.
- **Supplemental external demo** — public demo exists, but source is not represented as a current workspace in this repository.
- **Commercial / Acquisition asset** — packaged for sale, licensing, or transfer; does not imply verified traction.
