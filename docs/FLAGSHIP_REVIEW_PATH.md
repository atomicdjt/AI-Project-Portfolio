# Flagship Review Path

This portfolio is broad. Reviewers should not interpret every experiment as equally current, maintained, or relevant.

## Recommended 15-minute employer review

### 1. Portfolio Hub — orientation

Confirm the professional positioning, project hierarchy, canonical routes, and evidence labels.

### 2. BuildWorld AI — technical originality

Review deterministic graph simulation, reproducibility, scenario comparison, cascade analysis, SSI scoring, testing, and responsible-use limits.

### 3. ProcessHarbor — role alignment

Review SOP, onboarding, knowledge-base, documentation-gap, validation, versioning, and export workflows. Note the explicit boundary between the deterministic static demo and documented backend extensions.

### 4. WeaveStudio — productization and delivery

Review local-first workflow editing, recovery, portability, browser verification, acquisition packaging, and consent-gated provider assistance.

### 5. RedactReady Pro — privacy and human review

Review sensitive-information detection, evidence confirmation, redaction, reports, privacy boundaries, and claim-safe limitations.

## Supplemental work

ScamShield, FocusForge, GardenGrid, Amino Workbench, HearthLink, Mnemosyne, local-only applications, and documentation-first frameworks provide additional evidence. They are not intended to displace the flagship review path.

## Evidence rules

- A live URL proves availability at the time checked, not complete behavior.
- Tests prove only the behavior and environment they cover.
- Commercial packaging does not prove sales or demand.
- Legacy preserved deployments do not imply maintained source authority.
- AI-assisted implementation does not remove the need to demonstrate understanding, verification, and decision ownership.
