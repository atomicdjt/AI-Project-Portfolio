## Recommended Employer Review Path

This portfolio is designed to be evaluated quickly. The strongest employer evidence is concentrated in three projects:

1. **[BuildWorld AI](https://github.com/atomicdjt/buildworld-ai)** — most technically original implementation: deterministic graph simulation, cascade analysis, reproducible experiments, SSI scoring, local persistence, and reports.
2. **[RedactReady Pro](apps/redactready-pro-hri-os)** — strongest privacy and document-intelligence workflow: local analysis, sensitive-information review, risk scoring, evidence mapping, redaction, and export.
3. **[ProcessHarbor](apps/opspilot-ai-operations-toolkit)** — clearest role fit for technical operations, SOPs, onboarding, knowledge-base workflows, gap analysis, versions, and export bundles.

Then review **[ScamShield AI](apps/scamshield-ai)** for public-interest safety design and **[LayerForge Studio](apps/layerforge-studio)** for frontend and Canvas interaction depth.

This repository is evidence of practical initiative, AI-assisted execution, product judgment, documentation discipline, and the ability to turn ambiguous ideas into structured working systems. It is not presented as a claim of senior engineering tenure.

## Verification

Install with Node.js 22 or later:

```bash
npm ci
```

Primary repository checks:

```bash
npm run check:docs
npm run lint:apps
npm run typecheck:all
npm run test:all
npm run build:all
npm run verify
npm run verify:release
```

See [docs/verification.md](docs/verification.md) for exact scope, exceptions, public checks, and responsible interpretation of the results.

## Responsible Scope

- Local-first does not mean encrypted, durable, or compliance-certified storage.
- Deterministic scores and recommendations are product heuristics, not certified predictions or professional advice.
- AI-assisted output requires review and should not be represented as independently verified.
- Medical, legal, financial, fraud, infrastructure, and safety-critical decisions remain outside project scope unless a project explicitly documents a narrower educational use.
- Supplemental demos whose source is absent from this repository are labeled accordingly.
