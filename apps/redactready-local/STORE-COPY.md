## Short Description

A downloadable, MIT-licensed local-first privacy-review application with a curated source package, synthetic samples, buyer documentation, release provenance, and verification materials.

## Full Description

RedactReady Local helps users review files for potentially sensitive information before sharing them with AI tools, email recipients, clients, vendors, or public platforms.

The application runs its documented review workflow in the browser. Users can load supported files, review suggested findings, approve or reject detections, add manual redaction boxes, optionally run experimental OCR, complete a verification checklist, and export a reviewed copy and optional JSON workflow report.

### Important Product Boundary

RedactReady Local is an assistive privacy-review workflow. Automated suggestions, OCR, visual detection, and metadata handling can miss information or produce incorrect results. Users must manually inspect every source file and every exported file before sharing.

This product:

- does not guarantee complete detection;
- does not guarantee complete redaction;
- does not guarantee metadata removal;
- does not provide legal advice;
- does not certify HIPAA, FERPA, FOIA, GLBA, GDPR, or other compliance;
- does not replace professional or human review; and
- does not include managed hosting, custom integrations, or support commitments unless separately agreed.

### Local-First Does Not Mean Risk-Free

The application is designed not to intentionally send document contents to external servers during its documented browser workflow. Buyers are responsible for verifying their own deployment, browser behavior, network activity, hosting configuration, extensions, downstream systems, and final exports before using real information.

Use the included synthetic samples first. Do not upload real sensitive documents to a public demo unless you have independently verified the deployment for your intended use.

### Technical Requirements

This is a downloadable source-code implementation kit, not a managed SaaS subscription. Buyers should be comfortable following Node.js/npm setup instructions, deploying a Vite application, or working with a developer or AI coding assistant.

The base application does not require a paid API. Browser support affects OCR performance and QR/barcode detection.
