# HearthLink legacy migration

This directory preserves the static production artifact served by Netlify at `https://hearthlink-p2p-demo.netlify.app/` on July 16, 2026.

The original editable source project was not available locally during the migration. The files are retained as a legacy static artifact rather than represented as a maintained source implementation. Its canonical public demo is `https://ai-project-portfolio-hearthlink.vercel.app/`.

`config.js` intentionally enables demo mode and has no signaling URL. The deployment demonstrates the browser UI and local/offline behavior only; it does not provide a signaling server, live multi-peer rooms, or an operating P2P service.
