# Nexus Play

Nexus Play is a local digital gaming distribution platform demo built with React, Vite, and Express.

## Run

```bash
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5175/
```

The Express API runs on `http://127.0.0.1:3003/`.

## Features

- Storefront with generated hero key art.
- Search, genre filters, sorting, wishlist, cart, and simulated checkout.
- Demo account with wallet, rewards, owned library, and download queue.
- Production build served by the Express app.

## Scripts
