# Amino Acid Workbench legacy migration

This directory preserves the static production artifact served by Netlify at `https://aminoacidworkbench.netlify.app/` on July 16, 2026.

The original editable source project was not available locally during the migration. The `index.html` file is retained as a legacy static artifact rather than represented as a maintained source implementation. Its canonical public demo is `https://ai-project-portfolio-amino-workbenc.vercel.app/`.

The workbench remains an educational and research-triage tool. It is not clinical interpretation, diagnosis, treatment guidance, or an automated pathogenicity classifier.
