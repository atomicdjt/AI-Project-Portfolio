Product positioning

The strongest version of this product is not an automated pathogenicity classifier. It is an explainable teaching and research-prep environment that helps users ask better questions before deeper database review, conservation analysis, structural modeling, or expert interpretation.

For students

Learn how amino-acid chemistry, substitution matrices, and sequence context influence variant hypotheses.

For educators

Run guided cases, export classroom reports, and teach evidence separation without relying on black boxes.

For researchers

Batch-triage substitutions, document quality gates, and create reproducible evidence dossiers.

Student mode

Guided learning flow

  1. Observe the substitution. What changed: size, charge, hydrophobicity, aromaticity, or flexibility?
  2. Check the sequence. Does the reference amino acid match the provided protein sequence?
  3. Compare scoring panels. Is the result high because chemistry changed, evolution disfavors it, or both?
  4. Separate evidence types. Computed features are not the same as clinical assertions, population frequency, or experimental results.
  5. Write a hypothesis. State what might be disrupted and what evidence is still missing.
