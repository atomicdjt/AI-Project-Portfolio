ProcessHarbor Pro

ProcessHarbor Pro is a runnable operations documentation system for small businesses. It turns messy notes, FAQs, tickets, and policy fragments into structured SOPs, training checklists, knowledge-base articles, documentation gap reports, audit events, export bundles, and version history.

The public Vercel demo is deterministic-first and works without accounts, a database, or client-side secrets. The repository also documents a reference API, validation contracts, optional server-side AI design, and a database migration path; those provider-backed services are not represented as active on this static deployment.
