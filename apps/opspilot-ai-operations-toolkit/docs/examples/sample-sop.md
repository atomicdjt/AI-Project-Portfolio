# Sample SOP Export

This sanitized sample Markdown export demonstrates the ProcessHarbor Pro document output format for portfolio review.

## Purpose

Standardize a repeatable operations handoff.

## Steps

1. Log the request type and priority.
2. Verify required context.
3. Assign an owner and due date.
4. Summarize the next action.
5. Record completion and exceptions.

## Quality Checks

- Named owner present.
- Due date present.
- Escalation path present.
- Review cadence present.
