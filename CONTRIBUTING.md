# Contributing

This is primarily a personal portfolio repository. Public contributions are not expected, but corrections and responsible issue reports are welcome.

## Useful Contributions

- Broken link fixes
- Typo or documentation corrections
- Reproduction details for bugs in runnable apps
- Accessibility or security observations with synthetic examples

## Pull Request Guidelines

- Keep changes narrow and explain the user-facing impact.
- Do not add secrets, API keys, personal data, or sensitive documents.
- Do not replace or remove existing projects without prior discussion.
- Preserve the repository structure: runnable apps live in `apps/`, employer-facing writeups live in `projects/`, and shared review docs live in `docs/`.
- Run the relevant validation command before opening a PR. For broad changes, use `npm run verify`.
