# Code of Conduct

This portfolio is a professional review space. Everyone participating in issues, pull requests, discussions, and related channels is expected to communicate respectfully and constructively.

## Expected behavior

- Focus feedback on the work and its outcomes.
- Respect privacy, intellectual property, and confidential information.
- Help keep reports reproducible, specific, and safe to share.

## Unacceptable behavior

- Harassment, discriminatory language, threats, or personal attacks.
- Sharing personal, confidential, or security-sensitive information without permission.
- Misrepresenting demo capabilities, attempting to misuse demos, or submitting spam.

## Enforcement

The maintainer may edit or remove contributions, restrict participation, or report serious conduct as appropriate. For a private report or licensing question, contact the maintainer through the GitHub profile linked to this repository.

## Scope

This code applies wherever this repository is represented or discussed in an official project channel.
